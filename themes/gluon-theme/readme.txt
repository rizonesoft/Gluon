A modern, lightweight WordPress block theme built with Tailwind CSS v4.

== Description ==

Gluon is a clean, modern WordPress block theme designed for bloggers, businesses, and creative professionals. Built with Tailwind CSS v4 and Full Site Editing (FSE) support, it provides a solid foundation for building beautiful, accessible websites.

= Key Features =

* **Block Theme** - Full support for WordPress Site Editor and block templates
* **Tailwind CSS v4** - Modern utility-first CSS with OKLCH color palette
* **Accessibility Ready** - WCAG 2.1 AA compliant
* **Custom Colors** - Rizone Blue color palette with semantic color tokens
* **Two Column Layouts** - Flexible layout options for content and sidebars
* **Performance Optimized** - Minimal CSS and JavaScript
* **Translation Ready** - Fully internationalized

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the 'Add New' button.
2. Click 'Upload Theme' and choose the gluon.zip file.
3. Click 'Install Now' and then 'Activate'.

== Development ==

This theme uses Tailwind CSS v4 for styling. To modify styles:

1. Install dependencies: `npm install`
2. Watch for changes: `npm run watch`
3. Build for production: `npm run build`
